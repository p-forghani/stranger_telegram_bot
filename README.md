# Template Telegram Bot

Template to create telegram bots.


## How to Run:
1. add `src` to `PYTHONPATH`
```
export PYTHONPATH=${PWD}
```
2. Set the token as an environment variable running this command:<br>
```
export STRANGER_BOT_TOKEN=[your token]
```
3. Run the following:
```
python src/run.py
```
